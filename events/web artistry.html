Web Artistry

"Spells of code, elevating webs beyond imagination"


Welcome to **"Web Artistry"** where your skills in web development and creativity will take center stage! This event challenges you through two thrilling rounds that test both your coding expertise and design innovation.

Rounds Overview

  1. Round 1: "The Bootstrap Landing Page Challenge"

    Description:In this first round, you’ll utilize Bootstrap to create a responsive landing page. Each team will be assigned a unique theme for their landing page, and your goal is to design and develop an engaging, user-friendly page that fits the given theme.

  2. Round 2: "The Front end Challenge"

    Description:In the final round, selected teams will develop a full website using frontend technologies such as HTML, CSS, and JavaScript. You will receive a specific prompt/problem statement during the competition, which will guide the website development process. Your task is to create a fully functional, visually appealing website that meets the prompt’s requirements.

Rules & Regulations

  1. Each team must consist of 2-4 members.
  2. Teams will use Bootstrap to build a responsive landing page based on a theme provided to each team.
  3. Teams will develop a website using HTML, CSS, and JavaScript based on a prompt/problem statement given during the competition.
  4. All coding and development must be completed within the allotted time frame for each round.
  5. Teams must work collaboratively and distribute responsibilities evenly.
  6. The decisions of the event coordinators and judges are final and binding.
    7. Winners: The top 3 teams, based on performance, will receive awards.
